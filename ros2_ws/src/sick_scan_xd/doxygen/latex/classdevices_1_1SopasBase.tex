\doxysubsection{Detailed Description}
Class \mbox{\hyperlink{classdevices_1_1SopasBase}{Sopas\+Base}} encapsuls the communication to a sensor via Sopas\+Protocol. It offers the functions\+:
\begin{DoxyItemize}
\item invoke\+Methode
\item read\+Variable
\item write\+Variable
\item (un)register\+Event
\end{DoxyItemize}

Callback functions are used to inform you about incoming events (scans or eval cases).
