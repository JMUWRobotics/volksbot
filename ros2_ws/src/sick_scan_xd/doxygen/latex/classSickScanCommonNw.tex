\doxysubsection{Detailed Description}
Interface for T\+C\+P/\+IP. 

This class provides an interface for T\+C\+P/\+IP communication. It also contains simple methods for accessing the essential contents of the S\+O\+P\+AS message (for example, determining the payload and the S\+O\+P\+AS command used). It based on an example of S\+I\+CK AG.
